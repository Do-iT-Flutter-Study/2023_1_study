### 1주차 ~ 4주차: Dart & Flutter 기초 다지기
<a href="https://www.youtube.com/@codingchef/playlists"> 코딩셰프 님의 Flutter 강의 </a>를 바탕으로 기초를 다졌습니다.

  + 3/21
  + 무엇을 아는지 => 클래스, 함수, 자료형 등
  + 무엇을 모르는지 => Flutter로 개발을 어떻게 하는지, Dart의 Null Safety 등
